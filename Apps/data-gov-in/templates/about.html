A better way to visualize data.gov.in


The Problem

When when we went to the site http://data.gov.in/catalog, we realised that there were some shortcomings. Fo example, there is no way to quickly all the information published by the Government and zoom into your area of interest.

Here are some of the problems:

1. You can list only a few datasets at time.





2. You need to open a spreadsheet application like Excel to actually view the data.

The Solution

We have built a new site to browse the data based on the datasets published so far. Here are the highlights:

1. Home Page shows you the various sectors you can explore.



2. Each sector has its own page with a list of all the datasets.



3. No need to open different applications to view the data.

How we did it

1. Download all files from data.gov.in

We wrote a small script to scrape files and descriptions from data.gov.in and download all the files.

2. Convert all Excel (xls) files to (csv)

Using python library xlrd, we converted all the xls files to csv.

3. Classify the files based on keywords

Using basic keyword matching in titles and descriptions, we classified the files by sector.

4. Guess chart type and generate charts from data

Based on the data we generated charts using chart.js.

5. Make a web application and publish.

We made a basic web application using python (cgi), Twitter Bootstrap and JQuery to publish the data.
